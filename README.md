# NodeScraper
A Node Scrapper to Crawl STYFI Product Detail Page and check for discrepancies in product sizes.

# Technology Used

Node 8.11.1 <br>
Cheerio<br>
Request<br>
Fs<br>
Fast-Csv<br>

# To run the module 
 Open terminal  and following command <br>
 git clone https://github.com/madanpandey97/NodeScraper.git
 cd NodeScraper
 npm install 
 node scraper.js
